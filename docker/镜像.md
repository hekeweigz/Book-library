### 背景
docker镜像是最小的，被精简过的Linux系统，是不带vim命令的
> 使用命令 'vim a.txt' 进行新建编辑a.txt文件，就会提示找不到命令

### 给镜像容器新增vim命令

1. 更新镜像
> apt-get update

2. 下载vim功能
> apt-get -y install vim

3. 提交副本使成为一个新镜像
> docker commit -m="提交的描述信息" -a="作者" 容器id  要创建的目标镜像名:[标签名]
